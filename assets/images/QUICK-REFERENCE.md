## 🔗 URL Format

All images follow this format:
```
https://images.unsplash.com/[photo-id]?ixlib=rb-4.0.3&w=[width]&q=80
```

- **[photo-id]**: Unique Unsplash image identifier
- **[width]**: 1600 for full sections, 800 for cards
- **ixlib**: Unsplash library version
- **q**: Image quality (80)
