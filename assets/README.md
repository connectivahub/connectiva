## 🖼️ Image Usage

All images are currently loaded from Unsplash CDN:
- **Format**: `https://images.unsplash.com/[photo-id]?ixlib=rb-4.0.3&w=[width]&q=80`
- **Large sections**: 1600px width
- **Card images**: 800px width
- **Quality**: 80 (optimal balance of quality and file size)
